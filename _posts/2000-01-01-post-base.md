#### 1. 이미지 나타내기
이미지 저장할 경로는 '/assets/img/'<br>
저장할 파일명은 '[확장자를 제외한 포스트 파일명]-[숫자].webp'<br>

```bash
{% raw %}
{% include image-path.html cols=[나타낼 열 숫자] start=[가져올 이미지 순서] end=[가져올 이미지] %}
{% endraw %}
```
