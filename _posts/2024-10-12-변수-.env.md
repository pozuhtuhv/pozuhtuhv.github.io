## 변수 .env
`.env 파일 변수`

### 모듈 설치
```python
pip install python-dotenv
```

### .env 파일 내용
```python
USER = 'ABC'
```

### 실행 예시
```python
import os
from dotenv import load_dotenv

load_dotenv()

user = os.getenv('USER')
```
